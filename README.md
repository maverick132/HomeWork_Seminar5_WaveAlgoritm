# HomeWork_Seminar5_WaveAlgoritm
## Реализация волнового алгоритма
Реализован волновой алгоритм с разным методом обхода сосесдних ячеек (окрестности Мура и окрестности фон Неймана). 
Добавлено логирования для каждого типа поиска пути. 
Программа выводит в терминал первоначальное поле и поле с найденным путем, помечая ячейки знаком +.
